# 🐔 Sistema de Ventas de Presas de Gallinas

Aplicación web para registrar ventas de partes de gallina, controlar inventario, registrar clientes y generar reportes. Funciona completamente en el navegador usando HTML, CSS y JavaScript puro (sin backend).

## 🧩 Funcionalidades

- Registro de ventas agrupadas por cliente
- Control de inventario por tipo de presa
- Seguimiento de estado de pago (Pagado, Fiado, Pendiente)
- Soporte para múltiples líneas de venta
- Búsqueda por cliente
- Exportación a Excel
- Almacenamiento local (usa LocalStorage del navegador)

## 🚀 Uso

Solo abre `index.html` en tu navegador, o accede a la app desplegada:

🔗 [Ver aplicación en línea](https://halq676.github.io/ventas-gallinas/)

> Funciona desde celulares, tablets y computadoras de escritorio.

## 📁 Archivos

- `index.html`: Interfaz principal
- `style.css`: Estilos visuales
- `main.js`: Lógica completa de ventas, inventario y clientes
